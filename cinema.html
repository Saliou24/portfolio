Guide d'Installation et d'Utilisation du Backend du Laboratoire

Langage Utilisé : React

Le backend du Laboratoire est au format JSON utilisé dans le développement d'applications Web et d'API pour échanger des informations entre les serveurs et les clients.

Outils Utilisés :

Visual Studio Code pour éditer et gérer le code source.

Installation et Utilisation :

  1. Téléchargement des Fichiers : Tout d'abord, téléchargez les fichiers nécessaires à partir du lien fourni.

    2. laboratoire
    Backend Laboratoire
    Laboratoire Postman
  2. Ouvrir le Terminal : Pour lancer le backend, ouvrez un terminal sur votre ordinateur.
  3. Naviguer vers le Dossier de Téléchargement : Utilisez la commande cd pour naviguer vers le dossier où vous avez téléchargé les fichiers du Laboratoire. Par exemple :
    4. cd Downloads
  4. Lancement du Backend : Pour démarrer le backend, utilisez la commande java -jar Laboratoire.jar. Assurez-vous que Java est installé sur votre système. Cette commande exécute le fichier "Laboratoire.jar" qui contient l'application backend. 
    5. java -jar Laboratoire.jar
  5. Accéder au Backend : Une fois le backend lancé avec succès, il devrait être disponible à une adresse et un port spécifiques (par exemple, http://localhost:8080). Vous pouvez accéder à l'API du backend à l'aide d'un client API tel que Postman.
